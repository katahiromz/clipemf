------------------------------------------
          EMF作成ソフト ClipEMF
              by 片山博文MZ
------------------------------------------

☆概要

ClipEMFは、コピーされたワードアートやオートシェイプなどを
EMF（拡張メタファイル）形式のファイルとして保存するソフトです。

☆使い方

ClipEMFを起動すると、クリップボードの監視が始まり、ワードや
エクセルなどで図形がコピーされると、コピーされた図形を表示
します。その後、「ファイル」メニューから「EMFとして保存...」を
選ぶと、表示されている図形をEMF形式のファイルとして保存できます。

☆動作環境

 * Windows 98
 * Windows Me
 * Windows 2000
 * Windows XP
 * Windows Vista
 * Windows Server 2003
 * Windows 7
